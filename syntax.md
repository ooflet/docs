# Syntax
{: .no_toc }
Kommand is extremely flexible when it comes to inputting commands. Kommand will ignore characters other than the command and it's arguments.

```
command argument
```
&
```
command(argument)
```

will work both ways unless the command requires a string as it's argument, which it will capture all text other than the command.

Although you cannot run 2 commands at one time.
